# ppt_by_slidev
### introduction
A presentation generated by Slidev
### Getting Start
- 1.下载该仓库源码git clone https://github.com/MGuangYao/ppt_by_slidev.git
- 2.配置环境见 https://cn.sli.dev/guide/
- 3.使用vscode打开test文件
- 4.新建终端输入命令slidev即可自动运行，在终端输入o即可打开
- 5.修改请关注slides.md文件
- 6.修改时可使用slidev .\slides.md --remote可使用第二设备在同一局域网下实时展示修改效果
